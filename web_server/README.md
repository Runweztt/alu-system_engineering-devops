# Web Server

This project covers secure file transfers, Nginx installation, DNS
configuration, permanent HTTP redirection, and custom HTTP 404 pages.

## Learning objectives

- Understand the main role of a web server.
- Understand parent and child processes.
- Understand common HTTP requests.
- Understand DNS and common DNS record types.
- Transfer files securely using SCP.
- Install and configure Nginx.
- Configure permanent HTTP redirection.
- Configure a custom HTTP 404 response.

## Files

| File | Description |
| ---- | ----------- |
| `0-transfer_file` | Transfers a file to a remote server using SCP. |
| `1-install_nginx_web_server` | Installs and configures Nginx. |
| `2-setup_a_domain_name` | Contains the registered root domain. |
| `3-redirection` | Configures a permanent redirect. |
| `4-not_found_page_404` | Configures a custom 404 response. |
| `5-design_a_beautiful_404_page.html` | Creative custom 404 page. |
